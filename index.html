Open HMI Tester

"Open HMI Tester" is a GUI Testing tool following an open architecture that describes a non intrusive capture/replay tool based on GUI Events. It may be adapted to support different windowing systems and operating systems used in the testing environment.

Open HMI Tester

"Open HMI Tester" is a GUI Testing tool following an open architecture that describes a non intrusive capture/replay tool based on GUI Events. It can be adapted to support different windowing systems and operating systems used in the testing environment.

Please, visit the project webpage for updated info: OHT site
